# StudyVerse v1.0

A modern educational website built with React + Vite.

## Features

* Responsive Design
* Notes Section
* Quiz Section
* Mock Tests
* About Page
* Contact Page
* Progressive Web App (PWA)
* SEO Optimized
* Fast Loading

## Installation

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Technologies Used

* React
* Vite
* React Router
* React Icons
* PWA
